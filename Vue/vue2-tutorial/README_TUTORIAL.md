# Vue 2 教學專案

這個專案整理了所有 Vue 2 的核心概念與實作範例。

## 📚 教學章節

### 01. 起步
- Vue 介紹
- 資料綁定

### 02. 事件處理
- 事件的基本使用
- 事件修飾符
- 鍵盤事件

### 03. 計算屬性 (Computed)
- 姓名案例
- 天氣案例
- Computed 完整寫法與簡寫

### 04. 偵聽器 (Watch)
- Watch 基本使用
- 深度監聽
- Watch vs Computed

### 05. Class 與 Style 綁定
- Class 綁定（對象、數組）
- Style 綁定（對象、數組）

### 06. 條件渲染
- v-if / v-else-if / v-else
- v-show
- v-if vs v-show

### 07. 列表渲染
- v-for 基本使用
- key 的作用與原理
- 列表過濾
- 列表排序
- Vue.set 方法
- Vue 監測數據原理

### 08. 表單輸入
- v-model 基本使用
- 表單修飾符（.lazy, .number, .trim）
- 收集表單數據

### 09. 指令
- 內建指令（v-text, v-html, v-cloak, v-once, v-pre）
- 自定義指令（函數式、對象式）

### 10. 生命週期
- 生命週期鉤子函數
- 各階段的特點與用途

### 11. 組件
- 組件基本使用
- 組件嵌套
- Props 父傳子
- Emit 子傳父
- 插槽（Slot）

### 12. 過濾器
- 過濾器的定義與使用
- 時間格式化範例

### 13. 瀏覽器存儲
- LocalStorage
- SessionStorage

### 14. 過渡動畫
- Transition 組件
- 動畫效果

## 🚀 運行專案

```bash
# 安裝依賴
npm install

# 啟動開發服務器
npm run serve

# 構建生產版本
npm run build
```

## 📖 使用說明

1. 啟動專案後，訪問首頁查看所有教學章節
2. 點擊相應的卡片進入不同的教學主題
3. 每個主題都包含：
   - 實際演示
   - 代碼範例
   - 詳細說明

## 🎯 學習路徑建議

1. **基礎部分**：起步 → 資料綁定 → 事件處理
2. **響應式數據**：Computed → Watch
3. **模板語法**：Class/Style → 條件渲染 → 列表渲染
4. **表單處理**：v-model → 表單數據收集
5. **進階功能**：指令 → 生命週期 → 組件
6. **實用工具**：過濾器 → 本地存儲 → 動畫

## 📝 注意事項

- 本專案基於 Vue 2.x 版本
- 所有範例都可以直接在頁面上互動測試
- 建議按照章節順序學習，由淺入深
