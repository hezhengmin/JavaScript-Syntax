## 📋 使用說明

### 啟動 Vue 2 專案
```bash
cd vue2-tutorial
npm install          # 首次需要安裝依賴
npm run serve        # 啟動開發服務器
```
訪問：http://localhost:8080

### 啟動 Vue 3 專案
```bash
cd vue3-tutorial
npm install          # 首次需要安裝依賴
npm run dev          # 啟動開發服務器
```
訪問：http://localhost:5173
